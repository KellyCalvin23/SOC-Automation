# Screenshots

Drop image files here with the filenames below and they'll automatically show up in the main [README.md](../README.md), which already links to these paths.

| Filename | What to capture |
|---|---|
| `n8n-workflow-canvas.png` | The full n8n workflow canvas (Wazuh SOC Triage), zoomed out enough to show all five numbered stage sticky notes and the node chain. |
| `discord-alert-example.png` | A real Discord alert embed showing Level, Agent, Source IP, IP Reputation (AbuseIPDB), AI Summary, and Suggested Next Step all populated. |
| `wazuh-alert-source.png` *(optional)* | The matching raw Wazuh alert (e.g. from `alerts.json` or the Wazuh dashboard) that produced the Discord alert above, for a before/after comparison. |

PNG or JPG both work. No fixed size requirement, but ~1200-1600px wide reads well on GitHub without needing to click through.
